# Mecha

Mecha is an constructive solid modeling language embedded in [Haskell](http://haskell.org/).

Mecha renders models with [POV-Ray](http:/povray.org/).

## Links

- [Mecha Hackage Library](http://hackage.haskell.org/package/mecha)
